# Palindrome Checker in Python

## What it is:
### A simple program that checks if a string the user inputs is a palindrome(same front and back). Coded to eliminate any confusion with spaces as well.

# Features of this program:
1. Tells you whether a word or a phrase is a palindrom
2. You dont have to worry about spaces so you can input a phrase or multiple words as well
   
# How to run this program:
first copy the code given in the palindromechecker.py file and then paste it into Visual Studios Code or any IDE that supports C#.

#Techniques used to build this program:
1. if else statement
2. functions
3. variables
   
## This code works by taking in a phrase or a word from the user and reversing it to see if it is a palindrome.
